# REMO_OX Downloader 🚀

A modern, cross-platform video and audio downloader built with Python and PyQt5. It provides a user-friendly interface to download content from various platforms using the powerful `yt-dlp` engine.

## ✨ Features
* **Multi-language Support:** Full support for Arabic (RTL) and English (LTR) interfaces.
* **High Quality:** Options to download video up to 1080p or extract high-quality MP3 audio.
* **Smart Dependency Management:** Automatically detects and downloads `FFmpeg` if it's missing, ensuring seamless media conversion.
* **Modern UI:** Dark-themed, responsive interface with progress tracking.
* **Customizable:** Choose your preferred download directory and file quality.

## 🛠️ Built With
* **Python 3.x**
* **PyQt5** - For the Graphical User Interface.
* **yt-dlp** - The core downloading engine.
* **Requests** - For managing component downloads.
